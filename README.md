# 🌌 Acerca de Bingflix

**Bingflix** es una plataforma web enfocada en mostrar películas y series online mediante integración con la API de TMDB.

La aplicación permite navegar entre diferentes categorías multimedia utilizando sliders interactivos y una interfaz inspirada en servicios modernos de streaming.

La plataforma permite:

- 🎬 Ver películas populares
- 📺 Explorar series de televisión
- 🔥 Mostrar contenido top rated
- 🇰🇷 Ver series coreanas
- 🎞️ Explorar películas Bollywood
- 🎥 Visualizar trailers
- 📱 Navegación responsive
- ⚡ Interfaz dinámica

El proyecto fue desarrollado para practicar:

- HTML5
- CSS3
- JavaScript
- APIs REST
- TMDB API
- Responsive Design
- Frontend dinámico

# ⚡ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/PlataformaStreamingPeliculas.git
```

---

## 2️⃣ Configurar API KEY

Obtener una API KEY desde TMDB:

```bash
https://www.themoviedb.org/
```

---

## 3️⃣ Reemplazar TMDB_API_KEY

Editar archivo:

```bash
config.js
```

Agregar:

```js
const API_KEY = "TU_API_KEY";
```

---

## 4️⃣ Ejecutar proyecto

Abrir:

```bash
index.html
```

O usar:

- VSCode Live Server
- XAMPP
- Apache
- Navegador web
